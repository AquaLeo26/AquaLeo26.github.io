# Aqua Leo website

Public website for Aqua Leo Payment Processing Solutions, hosted free on GitHub Pages.

- `index.html` — home page (currently a temporary stand-in; replace with the real export)
- `signupnow.html` — merchant sign-up form (submissions emailed via FormSubmit to aqualeo@aqualeo.net)
- `thankyou.html` — post-submission confirmation
- `styles.css` — shared brand styles

This repo is intentionally **public** (required for free GitHub Pages) and contains no customer data — form submissions go straight to email and are never stored here.
